# 🧱 벽돌깨기 (Brick Breaker) Game Design Document

## 1. 개요 (Overview)

* **장르**: 아케이드 / 캐주얼
* **플랫폼**: 웹 (PC / 모바일), 향후 모바일 앱 확장 가능
* **타겟 유저**: 짧은 플레이를 선호하는 모든 연령대
* **플레이 타임**: 1판당 2~5분
* **핵심 재미**: 반사 각도 컨트롤, 파괴 쾌감, 점진적 난이도 상승

---

## 2. 게임 목표 (Game Objective)

플레이어는 패들(Paddle)을 좌우로 이동시켜 공(Ball)을 튕기며 화면 상단의 모든 벽돌(Bricks)을 제거해야 한다. 공이 화면 하단으로 떨어지면 라이프를 잃으며, 모든 라이프를 소진하면 게임 오버가 된다.

---

## 3. 핵심 게임 플레이 (Core Gameplay)

### 3.1 기본 규칙

* 공은 패들에 맞아 반사된다.
* 공이 벽돌에 닿으면 벽돌이 파괴되거나 내구도가 감소한다.
* 모든 벽돌을 제거하면 다음 스테이지로 이동한다.
* 공이 바닥 아래로 떨어지면 라이프 -1.

### 3.2 조작 방법

* **PC**: 마우스 이동 또는 ← / → 키
* **모바일**: 터치 드래그

---

## 4. 게임 요소 (Game Elements)

### 4.1 패들 (Paddle)

* 좌우 이동만 가능
* 아이템 효과에 따라 길이 변경 가능

### 4.2 공 (Ball)

* 일정 속도로 이동하며 충돌 시 반사
* 속도는 스테이지에 따라 증가

### 4.3 벽돌 (Bricks)

| 타입     | 설명            |
| ------ | ------------- |
| 일반 벽돌  | 1회 충돌 시 파괴    |
| 강화 벽돌  | 2~3회 충돌 필요    |
| 무적 벽돌  | 파괴 불가, 반사만 가능 |
| 아이템 벽돌 | 파괴 시 아이템 드롭   |

### 4.4 아이템 (Items)

* 패들 확장
* 공 추가 (멀티볼)
* 공 속도 증가 / 감소
* 레이저 패들 (선택)

---

## 5. 스테이지 디자인 (Level Design)

* 스테이지는 행(Row)과 열(Column) 기반의 벽돌 배열로 구성
* 스테이지가 진행될수록:

  * 벽돌 수 증가
  * 강화 벽돌 비율 증가
  * 공 속도 증가

---

## 6. 점수 시스템 (Scoring System)

* 벽돌 파괴 시 점수 획득
* 연속 파괴(콤보) 시 보너스 점수
* 남은 라이프에 따른 클리어 보너스

---

## 7. 게임 상태 (Game States)

* Boot / Preload
* Title
* Ready
* Playing
* Stage Clear
* Game Over
* Pause

---

## 8. UI / UX

### 8.1 UI 요소

* 현재 점수
* 남은 라이프
* 현재 스테이지
* 일시정지 버튼

### 8.2 피드백

* 벽돌 파괴 시 이펙트 & 사운드
* 공 반사 시 타격감 있는 효과음

---

## 9. 사운드 & 연출

* 배경 음악(BGM): 경쾌한 아케이드 스타일
* 효과음(SFX):

  * 공 반사
  * 벽돌 파괴
  * 아이템 획득
  * 게임 오버

---

## 10. 기술 스택 (Technical Stack)

* **엔진**: Phaser 3
* **언어**: TypeScript
* **빌드**: Vite
* **배포**: GitHub Pages

### 10.1 화면 사양

* **기본 해상도**: 480 x 800 (세로 모드, 9:16 비율)
* **스케일 모드**: FIT (비율 유지, 화면에 맞춤)
* **자동 센터링**: 화면 중앙 배치
* **반응형**: 브라우저 크기 변경 시 자동 스케일링
